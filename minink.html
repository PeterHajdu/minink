Introduction to Haskell

This course walks you through the basic language features in three weeks. A bite sized lecture is sent to you every day with exercises. If you need help you can simply reply to the email containing the lecture. Knowledge of an imperative language such as Java might help but is not necessary. All you need is a unix like operating system, 15-20 minutes every day and dedication. Enrollment is free but an invitation code is required. You can simply ask for one by contacting me.
